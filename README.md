## 🔨使用方法

1. 首先，你需要满足以下条件：

    * Nginx、Apache
    * PHP 7.4
    * Mysql 5.6+（推荐8.0）

2. 将下载好的源码压缩包上传至宝塔、虚拟主机中

3. 解压压缩包

4. 访问域名并开始安装过程

5. 安装结束后且后台配置完即可开始运营
